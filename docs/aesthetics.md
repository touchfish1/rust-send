# 美学设计 · 中国水墨风格

> 「墨分五色，计白当黑。」
> 以水墨画的美学原则指导 UI 设计：留白、克制、意境、自然。

---

## 1. 设计理念

### 核心美学原则

| 原则 | 水墨画概念 | UI 映射 |
|------|-----------|---------|
| **留白** | 疏可走马，密不透风 | 大面积负空间，元素间距充裕，不拥挤 |
| **墨分五色** | 焦浓重淡清，层次丰富 | 用灰度层次代替彩色堆叠，信息层级靠墨色区分 |
| **以形写神** | 不求形似求神似 | 图标/控件简约写意，不追求拟物 |
| **计白当黑** | 空白也是画面的一部分 | 空状态/留白区域视为设计元素而非"未完成" |
| **自然** | 浑然天成，不事雕琢 | 交互自然流畅，动画克制，不用夸张特效 |

### 不做什么

| 避免 | 原因 |
|------|------|
| 磨砂玻璃 / 毛玻璃效果 | 过于流行"AI味"，破坏纸墨质感 |
| 霓虹渐变 / 紫色系 | 科技感过重，与水墨冲突 |
| 超大圆角 (16px+) | 失去宣纸的方正质感，用 4-8px 微圆角 |
| 过度阴影 | 水墨讲平面层次，用墨色深浅代替投影 |
| 动态粒子 / 背景动画 | 破坏静谧感，分散注意力 |
| 聊天气泡 UI | 文件传输不是聊天，用卡片式布局 |

### 配色依据

```
宣纸白 #F5F0E8
  └ 传统宣纸本色，微黄温暖，不刺眼
  └ 区别于纯白 (#FFFFFF) 的科技感 / 医院感

墨色 #2C2A26
  └ 松烟墨色，微暖的黑，不采用纯黑
  └ 纯黑在纸上看太突兀，墨色有自然过渡

朱砂红 #C23A2B
  └ 传统印章 / 书画题跋用色
  └ 作为点缀色只用于关键操作，不滥用
  └ 传承"点朱"的仪式感

青绿 #5B7B5A
  └ 青绿山水画中的矿物颜料色
  └ 用于完成 / 正确状态，平和而不刺眼

赭石 #B8863A
  └ 传统国画赭石色
  └ 用于警告 / 注意状态
```

## 3. 字体设计

### 字体系列

```
UI 英文 / 数字: Inter
  └ 干净、现代、与水墨风格搭配不违和
  └ 字偶距 (letter-spacing) 适当放宽营造留白感

UI 中文:系统默认（PingFang SC / Noto Sans SC / Microsoft YaHei）
  └ 中文字体不宜过度设计，系统默认最稳妥
  └ 字号稍大一号保证可读性

文件名 / 代码: JetBrains Mono
  └ 等宽字体，适合文件列表中的名称对齐
  └ 用于技术细节，与主体形成微对比

App 名称 / Logo: 可考虑书法字体（如 站库朗宋 / 演示秋鸿）
  └ 仅用于启动页 / 关于页，UI 中不使用
```

### 字号层级

```
Hero / 空状态: text-4xl (36px)    ← 留白大字
H1: text-2xl (24px)                ← 页面标题
H2: text-xl (20px)                  ← 区块标题
H3: text-base (16px) + font-bold   ← 卡片标题
Body: text-sm (14px)                ← 正文
Caption: text-xs (12px)             ← 辅助 / 时间戳

行高: 1.6 (宽松，营造留白感)
段落间距: 1.5x 字号
```

## 4. 布局与间距

### 间距体系（8px 网格）

```
--space-1:  4px   (半个网格，用于图标与文字间距)
--space-2:  8px   (基础网格，组件内间距)
--space-3:  12px  (1.5x)
--space-4:  16px  (组件间距)
--space-6:  24px  (区块间距)
--space-8:  32px  (大区块间距)
--space-12: 48px  (页面级留白)
--space-16: 64px  (大幅留白)
```

### 布局比例

```
Sidebar:       w-72 (288px)
MainContent:   flex-1, p-8 (32px 四周留白)
Card 内边距:   p-5 (20px)
Card 间距:     gap-4 (16px)
列表项:        py-3 (12px 上下)
```

### 留白策略

```
页面边缘留白: 32px (像画作的裱边)
区块间留白:   32-48px (像画卷的段落间隔)
内容组内间距: 16-24px (像题跋与画面的距离)
元素间间距:   8-12px (适度的透气感)
```

## 5. 圆角与边框

### 圆角

```
--radius-none:  0px     (全局极少使用)
--radius-sm:    4px     (小元素 / 标签)
--radius-md:    6px     (按钮 / 输入框)
--radius-lg:    8px     (卡片 / 面板)
--radius-xl:    12px    (弹窗 / 对话框)

非对称圆角 —— 受国画折扇 / 卷轴启发：
  · 卡片左上角可略大 (8px)，右下角略小 (4px)
  · 按钮右上角可略大
  · 微妙的不对称增加手工感，但不影响可用性
```

### 边框

```
--border-width: 1px
--border-style: solid
--border-color: var(--border)

特殊场景：
  · 选中状态: border-color = var(--accent)，1.5px
  · 分割线: 使用 border-light，或干脆用留白代替
  · 避免用虚线（过于工业感）
```

## 6. 阴影与层叠

水墨风格弱化投影，用"墨色渗透"代替：

```
--shadow-sm: 0 1px 2px rgba(44, 42, 38, 0.04)    ← 极淡
--shadow-md: 0 2px 8px rgba(44, 42, 38, 0.06)     ← 轻
--shadow-lg: 0 4px 16px rgba(44, 42, 38, 0.08)    ← 中
--shadow-xl: 0 8px 32px rgba(44, 42, 38, 0.10)    ← 重

深色模式：
--shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.2)
--shadow-md: 0 2px 8px rgba(0, 0, 0, 0.3)
--shadow-lg: 0 4px 16px rgba(0, 0, 0, 0.4)

特点：
  · 不使用彩色阴影（如蓝色或紫色发光）
  · 阴影颜色取墨色 (--text) 而非纯黑
  · 阴影很淡，几乎不可见，只是微微分隔层级
```

## 7. 图标

### Lucide 配置

```
strokeWidth: 1.5    ← 比默认 2 更细，接近毛笔线条感
size: 16-20px       ← UI 图标
size: 24px          ← 空状态大图标
size: 32px+          ← 文件类型图标

不打折：
  · 不使用填充风格 (filled) 图标
  · 不使用彩色图标
  · 图标颜色继承 text-secondary
  · 仅关键操作图标使用 accent 色
```

### 文件类型图标

文件图标不采用 Material 风格的大块彩色卡片，而是：

```
轻量线稿风格：
  ┌──────┐
  │ □    │    ← 简洁的文档轮廓
  │      │
  │      │    线条极简，如白描
  └──────┘

不同格式只通过角标 / 线条密度微区分：
  PDF:    右下角小折页 + 极淡底色
  图片:   内置一个菱形（如山水画中的印章）
  视频:   右侧播放三角
  压缩包: 两道横纹

颜色使用 text-secondary，只在选中时变色。
```

## 8. 动画与过渡

### 原则

```
克制 —— 动画不是装饰，是功能的反馈

时长:     150-200ms（短促有力）
缓动:     ease-out（自然停止）
位移:     ≤ 8px
缩放:     ≤ 1.02x
不透明度: 0 → 1
```

### 具体动画

```
页面切换:    fade + translateY(4px), 200ms, ease-out
元素出现:    fade + translateY(2px), 150ms, ease-out
  
悬停:       背景色过渡 150ms, 不使用放大 / 发光
按下:       scale(0.98), 100ms
选中:       背景色过渡 200ms

进度条:      线性过渡, 300ms, 避免跳动
通知:        slide-in-right 250ms, 停留后 slide-out

不做:
   · 页面间滑动切换
   · 弹性动画 (spring)
   · 旋转 / 翻转
   · 连续循环动画（除非加载态）
```

## 11. 设计验证

### 检查清单

- [ ] 页面上是否同时出现了超过 3 种彩色？（应该只有朱砂红 + 一个语义色）
- [ ] 是否有不必要的渐变 / 发光 / 投影？（应该极克制）
- [ ] 中英文字体搭配是否协调？
- [ ] 留白是否足够呼吸？（每个区块边缘至少 24px）
- [ ] 空状态是否经过设计？（不是空白一片，而是有意境的白）
- [ ] 暗色模式的底色是否为纯黑 #000？（应该用暖灰）
- [ ] 图标是否全部为线稿风格？（无填色图标）
- [ ] 按钮文字是否为纯白？（应该是宣纸色 #F5F0E8 或类似暖白）

### 配色对比度

```
浅色模式:
  宣纸白 #F5F0E8 + 浓墨 #2C2A26 → 对比度 ~12:1 ✓
  宣纸白 #F5F0E8 + 淡墨 #8A8078 → 对比度 ~5:1  ✓（仅辅助文字）
  朱砂红 #C23A2B + 宣纸白        → 对比度 ~4.5:1 ✓（符合 WCAG AA）

深色模式:
  墨底 #1A1714 + 宣纸色 #D4CCC0  → 对比度 ~12:1 ✓
```
